# **Python Practice Projects — Day 1 (Foundational Exercises & Mini‑Projects)**

A collection of beginner‑friendly Python exercises designed to strengthen your fundamentals through **hands‑on coding**.  
This repository includes multiple small programs covering temperature conversion, list processing, string formatting, calculators, and a complete mini‑project: **Expense Logger**.

---

## **What’s Inside**

### **1. Temperature Converter**
Convert temperatures between Celsius and Fahrenheit using clean, reusable functions.

- **Celsius → Fahrenheit**
- **Fahrenheit → Celsius**  
- Includes error handling and a menu‑driven interface.

---

### **2. List Processor**
Work with lists of numbers to compute:

- **Maximum value**
- **Minimum value**
- **Sum**
- **Average**

Great for practicing Python’s built‑in functions and custom logic.

---

### **3. String Formatter**
Take a user’s full name and generate:

- **Uppercase**
- **Lowercase**
- **Reversed string**
- **Initials**  
  Example: *Saif ur Rehman → S.U.R.*

---

### **4. Simple Calculator**
Perform basic arithmetic using functions:

- **Addition**
- **Subtraction**
- **Multiplication**
- **Division (with zero‑check)**

Includes input validation and a loop‑based menu.

---

### **5. Multi‑Number Calculator**
A more advanced calculator that accepts **multiple numbers** and performs:

- **Add**
- **Subtract**
- **Multiply**
- **Divide**

Supports comma‑separated or space‑separated inputs.

---

### **6. Mini‑Project — Expense Logger**
A complete CLI application that allows users to:

- **Add expenses**
- **View expenses**
- **Calculate total spent**
- **Save to file**
- **Load from file**

Data is stored in `expenses.txt` and loaded automatically.
